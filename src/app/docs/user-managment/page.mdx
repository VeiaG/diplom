# Реєстрація користувачів

![Реєстрація користувачів](/docs/13.png)

На цій сторінці адміністратор системи може реєструвати нових користувачів у системі, а також керувати існуючими обліковими записами.
В системі є два типи користувачів: адміністратори та звичайні користувачі.

- **Адміністратори**: мають повний доступ до всіх функцій системи, включаючи управління користувачами, налаштування системи та доступ до всіх даних.
- **Звичайні користувачі**: мають обмежений доступ до функцій системи (Усі функції , окрім адміністрування).

Також для кожного користувача можна вказати його підрозділ , який буде використовуватись для стандартної фільтрації по розділам на головній сторінці системи.
