# Імпорт та Експорт

![Імпорт та Експорт](/docs/19.png)

В системі реалізовано імпорт та експорт усіх даних БД в форматі Excel (xlsx).
Для імпорту даних в систему потрібно спочатку завантажити шаблон для іморту ( який водночас є експортом даних з системи), який налаштований під структуру даних системи. 
Після заповнення шаблону, його потрібно завантажити в систему використовуючи відповідну кнопку імпорту та попередньо вибравши файл для імпорту.

## Шаблон імпорту

![Шаблон імпорту](/docs/20.png)

Під час завантаження шаблону імпорту, система автоматично генерує файл з даними, які вже є в системі.
Різні таблиці бази даних йдуть на різні аркуші в ексель файлі. Для полів з зовнішніми ключами в шаблоні імпорту, уже налаштовані випадаючі списки з даними , які вже є заповнені в таблицях.
Це дозволяє легше заповнити шаблон імпорту.

> **Увага**: Не пропускайте рядки в таблиці , адже система імпортує дані по рядках , що може призвести до неочікуваних наслідків. 
> Також рекомендується створити дамп бази даних перед імпортом даних, щоб мати можливість відновити дані в разі будь-якої помилки.
