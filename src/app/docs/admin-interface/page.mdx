# Огляд інтерфейсу адміністратора

![Інтерфейс адміністратора](/docs/12.png)

Ця сторінка призначена для адміністраторів системи і забезпечує інтерфейс для повного управління всією системою.

## Структура інтерфейсу

Інтерфейс розділений на дві основні зони: бічна навігаційна панель зліва та основна робоча область контенту справа. 
В основному вони потрібні для навігації між різними функціональностями адміністрування системи.


- **Назад до застосунку**: Посилання для повернення до основного інтерфейсу користувача програми.
- **Розділ "Управління застосунком"**:
    - **Адмін панель**: Головна сторінка адміністративного інтерфейсу.
    - **Користувачі**: Перехід до інтерфейсу керування обліковими записами користувачів.
    - **Звіти**: Генерація загального звіту про систему. (Усі дані з бази даних)
- **Розділ "БД та генерація"**:
  - **Редагувати таблиці**: Перехід до інструментів для редагування структур таблиць бази даних.
  - **Контракти**: Доступ до управління шаблонами та екземплярами контрактів.
  - **Імпорт даних**: Перехід до функціоналу імпорту та експорту даних.
