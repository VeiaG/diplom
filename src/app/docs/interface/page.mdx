# Огляд інтерфейсу

![Огляд інтерфейсу](/docs/1.png)

Ця сторінка призначена для користувачів системи і забезпечує інтерфейс для перегляду, пошуку та фільтрації списку вступників.

## Функціональність

- **Пошук вступника:** текстове поле дозволяє виконати пошук за ПІБ вступника в реальному часі.
- **Фільтрація:** кнопка "Фільтри" відкриває доступ до розширених фільтраційних опцій 
- **Скидання фільтрів:** кнопка "Скинути фільтри" очищає всі активні фільтри.
- **Додавання нового запису:** кнопка "+" відкриває форму для створення нового запису про вступника.

| Колонка          | Опис                                                         |
| ---------------- | ------------------------------------------------------------ |
| ПІБ вступника    | Повне ім’я вступника                                         |
| ПІБ представника | Повне ім’я особи, що представляє вступника                   |
| Підрозділ        | Назва факультету або інституту                               |
| Спеціальність    | Код та назва спеціальності                                   |
| Фінансування     | Тип фінансування: бюджет або контракт                        |
| Ступінь          | Освітній рівень (наприклад, доктор філософії)                |
| Дата створення   | Дата створення запису                                        |
| Дата оновлення   | Дата останнього оновлення запису                             |
| Меню дій         | Кнопка з трьома крапками (`⋯`) для доступу до додаткових дій |

Більше колонок можна показати, натиснувши на кнопку з іконкою ока. Це відкриває меню, де можна вибрати, які колонки відображати.

## Редагування та завантаження сформованих документів

<img src="/docs/4.png" alt="Редагування та завантаження документів"  />

Для завантаження документів , або редагування даних про вступника, потрібно натиснути на кнопку з трьома крапками (`⋯`) у колонці "Меню дій". Це відкриває меню з такими опціями:

- **Завантажити контракти** - завантажити сформовані документи для вступника , всі документи зберігаються в zip-архіві.
- **Редагувати** - відкрити форму редагування даних про вступника.
-  **Видалити** - видалити запис про вступника. 
> Увага! Видалення запису є незворотнім процесом. Після видалення дані не можуть бути відновлені.

## Фільтрація 

Реалізовано фільтрацію за такими параметрами:

- **Підрозділ** - вибір з доступних навчальних підрозділів.
- **Спеціальність** - вибір з доступних спеціальностей.
- **Фінансування** - вибір з доступних типів фінансування.

## Сортування та налаштування колонок

<img src="/docs/3.png" alt="Сортування"  />

Кожна колонка таблиці може бути просортована за зростанням або спаданням. Для цього потрібно натиснути на заголовок колонки. 
Також в меню заголовка можна приховати колонку , або перемістити її в інше місце.
